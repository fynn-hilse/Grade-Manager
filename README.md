# Grade-Manager
Ein einfaches CLI Java-Tool zur Noten- und Namenverwaltung von Studenten. 

#Features 
-Eingabe von Noten mit Validierung
-Berechung des Notendurchschnitts 
-Finden der besten Note
-Sortierung der Ergebnisse (Bubble Sort) incl. Check, ob Array schon sortiert ist
