# Grade-Manager
Ein einfaches CLI Java-Tool zur Noten- und Namenverwaltung von Studenten. 

#Features<br>   
-Eingabe von Noten mit Validierung<br>
-Berechung des Notendurchschnitts<br>
-Finden der besten Note<br>
-Sortierung der Ergebnisse (Bubble Sort) incl. Check, ob Array schon sortiert ist<br>
